# dropout-academic
O projeto consiste em uma análise situacional de alunos ingressantes na Universidade Federal de Pernambuco. Isso será feito a partir da extração de informações gráficas, principalmente, sobre a evasão escolar no ambiente acadêmico, a fim de obter conhecimentos para possíveis tomadas de decisão acerca de situações acadêmicas de alunos matriculados.   Dessa forma, será desenvolvido um site contendo essas visualizações, dentre elas a evolução - ou possível sazonalidade - do nível de evasão escolar, vinculada às informações de curso, benefícios recebidos e dados geográficos.
